## 🚀 Featured Projects

### 🔬 [BubbleID Deep Learning Framework](https://github.com/david1-max/BubbleID-Reproduction)
A reproduction of the BubbleID deep learning framework designed to automate pool boiling heat transfer analysis. 
* **Key Features:** Mask R-CNN boundary segmentation, OC-SORT object tracking, and local interface velocity mapping using cKDTree nearest-neighbor boundary tracking.
* **Tech Stack:** Python, PyTorch, Detectron2, OpenCV, SciPy.

### 💬 [VibeChat](https://github.com/david1-max/vibe-chat)
A low-latency, real-time web application supporting messaging and video calls.
* **Key Features:** WebSockets for instant text messaging and signaling, and WebRTC for direct browser-to-browser audio/video streams.
* **Tech Stack:** Node.js, Express, WebSockets, WebRTC, Vanilla JS.

### 🏛️ [JanSahayak AI](https://github.com/david1-max/JanSahayak-) (Under Construction)
An AI-powered recommender platform matching students with personalized government schemes based on their profile criteria.
* **Key Features:** Dynamic profile parsing and semantic matching algorithm.
* **Tech Stack:** Python, Flask, Pandas.
